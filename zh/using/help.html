Getting Help #

    我们总是很乐意帮助您解决 Omnivore 遇到的任何问题或您的任何反馈。

    • Discord: 获得帮助的最佳方式是 加入我们的 Discord 并在 #help 频道中寻求帮助。
    • 电子邮件: 给我们发送电子邮件至 feedback@omnivore.app
    • 聊天: 使用应用菜单中的“反馈”菜单项访问我们的支持聊天。
