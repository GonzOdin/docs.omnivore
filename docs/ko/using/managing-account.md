Omnivore 계정과 저장소 관리를 위한 추가 도구가 제공되고 있습니다.

[[toc]]

## 계정 삭제

계정을 삭제하기 위해서는 [계정 삭제 도구](https://omnivore.app/settings/delete-my-account)를 이용해 주세요.

## 저장소 전체 보관

저장소에 있는 모든 문서를 일괄적으로 보관함에 옮기려면 [일괄 수정 도구](https://omnivore.app/tools/bulk)를 이용해 주세요.
